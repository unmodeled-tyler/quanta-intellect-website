Vessel Windows Installer Return Codes

Last updated: April 8, 2026

Vessel Browser uses a standard NSIS-based Windows installer. The installer currently uses the standard NSIS return codes and does not intentionally define additional custom miscellaneous EXE return codes.

Standard Return Codes

  • 0 — Normal execution (success)
  • 1 — Installation aborted by the user
  • 2 — Installation aborted by the installer script

Silent Installation

Vessel supports standard NSIS silent installation with /S.

Example: Vessel-0.1.3-x64-setup.exe /S

Notes

If future Vessel releases introduce custom installer logic or additional return codes, this page will be updated accordingly.

Reference

NSIS standard error levels are documented by NSIS in Appendix D: https://nsis.sourceforge.io/Docs/AppendixD.html
